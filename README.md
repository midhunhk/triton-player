Triton Player
=============

Triton Player is an MP3 player based on Adobe AIR platform. 

Features
========
* Brilliant UI (transparent)
* Different Visualization options
* Create and load playlists
* Multilingual (Implementation for English and German)
* Shuffle playlist
* Supports Themes
* Seekbar
